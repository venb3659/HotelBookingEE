1. Download the Automation Eclipse Workspace items from GIT
2. To run this project, please ensure the latest version of Firefox browser, Java, Eclipse & TestNG plug In are installed
3. Launch Eclipse with the Workspace location pointing to the downloaded location
4. Right click on the TestNG.xml file in Eclipse and select Run As > TestNG Suite
5. This will run the 2 test cases with 2 sets of test date in Firefox browser
